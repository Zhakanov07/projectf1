Как деплоить проект на Vercel

1) Через GitHub (рекомендуется):
   - Перейдите на https://vercel.com, войдите и выберите "Import Project" → "From Git Repository".
   - Подключите репозиторий `https://github.com/Zhakanov07/projectf1.git` и выберите ветку `main`.
   - Оставьте поля сборки пустыми (статический сайт). Нажмите Deploy.

2) Через Vercel CLI (локально):
   - Установите CLI: `npm i -g vercel`.
   - В корне проекта выполните `vercel` и следуйте подсказкам для привязки проекта.

Примечания:
- Конфигурация хранится в `vercel.json`.
- Убедитесь, что ненужные файлы перечислены в `.vercelignore`.
